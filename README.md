# 机械制图智能助手 (MechDraw Tutor) - 项目使用说明书

欢迎使用“机械制图智能助手”！本项目是一个专为中职学生和教师设计的交互式教学演示平台，结合了动态 SVG 绘图演示和 DeepSeek 智能 AI 问答功能。

## 1. 项目简介

本应用旨在解决机械制图教学中的痛点：
*   **可视化演示**：通过动态步骤展示圆弧连接、三视图投影等复杂作图过程。
*   **AI 智能辅导**：内置 DeepSeek 大模型，能够像助教一样随时解答关于国标规范、绘图步骤的疑问。
*   **教师演示工具**：清晰的界面和操作流程，适合在多媒体教室进行投屏讲解。

## 2. 环境准备 (Prerequisites)

在使用本系统前，请确保您的电脑上已经安装了以下软件。如果没有，请点击链接下载安装。

1.  **Node.js**: 运行 JavaScript 环境的基础。
    *   下载地址: [https://nodejs.org/zh-cn/](https://nodejs.org/zh-cn/)
    *   *推荐版本*: LTS (长期支持版)，例如 v18.x 或 v20.x。
2.  **Git**: 用于下载项目代码。
    *   下载地址: [https://git-scm.com/](https://git-scm.com/)

## 3. 安装步骤 (Installation)

请按照以下步骤在您的电脑上部署本项目：

**步骤 1：下载代码**
打开终端（Windows 用户请搜索 "cmd" 或 "PowerShell"，Mac 用户打开 "Terminal"），输入以下命令：

```bash
# 假设您已将代码解压或克隆到本地
# 如果是压缩包，请先解压，然后进入文件夹
cd mech-draw-tutor  # 进入项目文件夹
```

**步骤 2：安装依赖**
在项目文件夹内，输入以下命令来安装所需的第三方库（这可能需要几分钟）：

```bash
npm install
```
*如果下载速度慢，可以使用淘宝镜像：`npm install --registry=https://registry.npmmirror.com`*

## 4. 运行项目 (Running)

安装完成后，输入以下命令启动网页：

```bash
npm start
```

*   启动后，浏览器通常会自动打开。
*   如果没有自动打开，请手动在浏览器地址栏输入：`http://localhost:3000`

## 6. 功能使用指南

### 6.1 教学演示 (Demo)
1.  在左侧菜单栏点击“教学演示”。
2.  选择您想讲解的模块（例如：圆弧连接：两直线）。
3.  **自动播放**：点击右下角的蓝色“自动播放”按钮，动画将按步骤执行。
4.  **交互讲解**：您也可以直接点击画布上的线条或圆圈，系统会跳转到对应的绘图步骤，并高亮显示。

### 6.2 AI 智能提问
1.  点击屏幕右下角的蓝色悬浮按钮 **"AI 提问"**。
2.  在输入框中输入问题，例如：
    *   “怎么判断三视图的长对正？”
    *   “外切圆弧连接的圆心怎么找？”
3.  等待几秒钟，DeepSeek 智能助手将会根据国家标准给出回答。

### 6.3 查看说明书
1.  点击左侧菜单栏的“使用说明”。
2.  这里包含了设计思路、教学痛点分析以及工作流程图，适合在比赛答辩时展示。
